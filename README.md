# n-qubit Quantum Fourier Transform

Creation of a quantum circuit in Qiskit that implements a Quantum Fourier Transform on an arbitrary number of qubits defined by the user.

---

## Overview

This project implements a Quantum Fourier Transform on an arbitrary number of qubits (user-defined)

Quantum Fourier Transforms are a crucial component of Shor's algorithm, the driving force behind quantum computing's advantage over classical computers to encrpyt and decrypt data.

---

## Methodology

1. **Creation of Quantum Circuit with N qubits and N classical bits**
2. **Initialization of qubit state which we'll take the QFT of**
3. **Qubit state can be fed into the QFT function to either take the transform or inverse transform of the state**
   
---

## Skills Demonstrated

- Pythonic implementation of Qiskit code
- Dynamic definition of Quantum computing functions
- Understanding of Qiskit syntax

## Notes

This code was developed as part of research into unconventional superconductivity and is intended as a demonstration of numerical modeling techniques rather than a production software package.
